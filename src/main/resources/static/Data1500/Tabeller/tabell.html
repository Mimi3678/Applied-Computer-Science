Læringsmål

  • Forstå hvordan tabeller er bygd opp i kolonner og rader, hvilken rolle datatyper, primærnøkler, fremmednøkler.
  • Kunne bruke SQL for å definere tabeller med datatyper, primærnøkler, fremmednøkler og forretningsregler.
  • Kunne bruke SQL for å jobbe med innholdet i tabeller, det vil si sette inn, endre og slette rader
  • Kunne lage SQL-skript som oppretter tabeller og setter inn data.

    • En database inneholder mange tabeller

  • Figuren viser logisk datamodell for hobbyhuset
  • Hver boks tilsvarer en tabell
  • PK = Primærnøkkel (primary key)
  • FK = Fremmednøkkel (Foreign key)
  • Figuren nedenfor kan også ses på som en relasjonsdatabase, som er en samling av tabeller som henger sammen

    • Hva er SQL?

  • SQL er et språk som er spesielt utformet for arbeid med databaser
  • SQL kan brukes for å lage spørringer som henter ut data fra tabeller
  • SQL kan brukes til mange forskjellige databaseoppgaver.
  • Det finnes kommandoer for å lage nye tabeller, for å legge til, endre og slette data og for å også tildele brukere rettigheter mot databasen
